 Weather App

A modern and responsive weather application that provides real-time weather data for any city in the world.

About the Project:

This project is my first experience working with APIs.
It allows users to search for a city and view current weather conditions along with a short forecast.

Through this project, I practiced:

Fetching data from an external API
Working with asynchronous JavaScript (async/await)
Dynamically updating the UI
Creating responsive layouts

Technologies Used:
HTML5
CSS3
JavaScript (ES6+)
Weather API (wttr.in)
